## ADA Handle 是什麼
ADA Handle 是一套 Cardano 上的人類可讀命名系統，類似以太坊的 ENS，可以把不便於人類閱讀的長地址轉換成 $ 開頭的自定義名稱(例如`$imyimang`)，讓他人直接輸入你的 Handle 名稱就能進行轉帳
但 ADA Handle 和 ENS 系統不同，ADA Handle 是 Cardano 上的 NFT，透過定位 Handle 的持有者來解析地址，因此不需要年費或續約，只要錢包持有 ADA Handle 就會自動對應到錢包地址

## 如何獲得 Handle
### 購買現有 Handle
由於 ADA Handle 本質上是一個 NFT，所以你可以到 Cardano 最大的 NFT 交易平台 JPG.store 看看有沒有你喜歡的 Handle
進入 JPG.store 找到 [ADA Handle](https://www.JPG.store/collection/adahandle)，就能看到正在出售的 Handle，連結錢包後就能購買
![alt text](images/20251122/image.webp)

### 鑄造新 Handle
如果你沒看到想要的 Handle，想要自己鑄造一個全新的 Handle，可以到 ADA Handle 的[官網](https://handle.me/)，連結錢包後在中間輸入你想要的 Handle 名稱，下面就會顯示價格
![alt text](images/20251122/image-1.webp)
Handle 的定價規則如下
- 傳奇（ 1 個字符，共 39 個）- 僅限拍賣
- 超稀有（ 2 個字符，共 1,521 個）- 500 美元 ADA
- 稀有（ 3 个字符，共 59,319 個）- 100 美元 ADA
- 普通（ 4 - 7 個字符，共 140,842,288,080 个）- 50 美元 ADA
- 基本（8-15個字符，共 753,789,555,901,817,000,000 個）- 10 美元 ADA

決定好名字後選擇 Mint Handle 並簽署交易就好了，等待幾分鐘後 Handle 就會出現在你的錢包
![alt text](images/20251122/image-2.webp)

## 如何個人化 Handle
買完之後 handle.me 右上角就會顯示你的 Handle 了，點擊右上角選擇 **Personalize**
![alt text](images/20251122/image-3.webp)

在 Personalize 頁面你就能個人化自己的 Handle 了，可以選擇更改字型大小、設定社群媒體、PFP 或背景等
PFP 和背景都能設定成持有的 NFT，但是只有少數允許的項目，例如 SpaceBudz, Goats, Clay, Hosky 等

你也可以直接在背景選擇 Marketplace 直接購買可以當背景的 NFT
![alt text](images/20251122/image-4.webp)

修改完成後點擊右上角的 **Checkout** 然後結帳簽署交易就完成了
![alt text](images/20251122/image-5.webp)

你可以到 handle.me/handle名稱 查看完整的 Handle 頁面(這個頁面排版也可以客製化)
![alt text](images/20251122/image-11.webp)

## 如何使用 ADA Handle
目前大部分的 Cardano 錢包和 DApp 項目都支援 ADA Handle 的地址解析，中心化交易所則大部分不支援
![alt text](images/20251122/image-6.webp)
錢包轉帳時輸入 Handle 就能直接解析到錢包地址，錢包裡也會顯示 Handle
![alt text](images/20251122/image-7.webp)

以 Eternl 為例，發送資金時只要在地址欄位輸入 Handle 名稱(例如`$imyimang`)，他就可以自動解析 Handle 的地址並發送資金
![alt text](images/20251122/image-8.webp)

大部分 DApp 例如 JPG.store 或 Minswap 等也都支援 Handle
![alt text](images/20251122/image-9.webp)
![alt text](images/20251122/image-10.webp)
